## 私について

蘇浩南（**Haonan Su**）です。フルスタックのソフトウェアエンジニアとして活動しています。2022年に北京大学でコンピューターサイエンスの学士号を取得し、現在は南カリフォルニア大学の修士課程で学んでいます。使いやすく親しみやすいアプリケーションを設計し、形にすることに情熱を注いでいます。

## 研究

北京大学のオープンソースソフトウェアデータ分析研究室でインターンを経験し、GitHubのGood First Issueが新規コントリビューターに与える影響を研究しました。また、オープンソースへの参加を始める人に適したGood First Issueを提案する[GFI-Bot](https://gfibot.io)の開発にも携わりました。

この成果は「[GFI-bot: Automated Good First Issue Recommendation](https://doi.org/10.1145/3540250.3558922)」として、ESEC/FSE '22 Demo Trackで発表されました。

## 職歴

#### Amazon Web Services（AWS）（2025.5 - 2025.8）

社内Wikiページの生成と更新を自動化するシステム、AWS Wikif.aiを設計・開発しました。

- Electron、React、TypeScriptを用いたデスクトップアプリケーションを開発。LangGraphでLLMを活用したハイブリッドワークフローとエージェントを統合し、AWS Bedrockによるインテリジェントな文書生成を実現しました。
- 30以上の社内データソースを統合し、バックグラウンドでの差分更新に対応。文書の作成・保守にかかる時間を数時間から数分へ短縮し、複数のチームによるテストにつなげました。

Sohuでは、応用AIとプロダクト開発の最前線にある複数のプロジェクトを企画・開発しました。
