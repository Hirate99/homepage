## 工作经历

#### Amazon Web Services（AWS）（2025.5 - 2025.8）

设计并构建 AWS Wikif.ai，用于自动生成和持续更新内部 Wiki 页面。

- 使用 Electron、React 与 TypeScript 构建桌面应用；通过 LangGraph 编排混合式 LLM 工作流与智能体，并接入 AWS Bedrock 生成文档。
